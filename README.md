# SCHOOL INVEST TRACKER

## Installation
**Requirement: Docker** - _if you don't have docker, download it [here](https://www.docker.com/products/docker-desktop/)_

    - With makefile:
        - run `make`
    - Without makefile:
        - run `docker network create -d bridge sit-network`
        - run `docker compose up -d`
___
